Git

  • git add -A: Stage changes to commit all files
  • git commit -m "added CSS to project": commit them to the repository
  • git pull origin main: make sure our current branch is up to date before opening a pull request
  • git push origin feature/add-css: push up all of our changes
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • Control flow: the order in which code is executed, from top to bottom in JavaScript. Can be interrupted with conditional statements and loops
  • A for loop contains three important steps:
    1. starting point. declare a variable x and give it a value of 0.
    2. condition: as long as the number in the variable x is less than the length of the array, the loop will keep running.
    3. final statement allows the array to iterate over each item. x++
